# Temel Grafik Tasarım Uygulamaları

Bu repoda lise döneminde grafik tasarım dersleri kapsamında yaptığım temel seviye tasarım uygulamaları bulunmaktadır.

## Kullanılan Araçlar

- Adobe Photoshop
- Adobe Illustrator

## İçerik

- Temel tasarım çalışmaları
- Uygulama denemeleri
- Grafik tasarım pratiği çalışmaları

## Amaç

Grafik tasarım derslerinde temel araçları öğrenmek ve öğrendiklerimi pekiştirmek için pratik yapmaktır.
